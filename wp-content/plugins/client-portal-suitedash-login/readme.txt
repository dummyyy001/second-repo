== Description ==

<h3 style="margin-bottom:20px;"><a href="https://suitedash.com">SuiteDash : All-in-One Business Software</a></h3>
<p>Visit Website &gt;&gt; <a href="https://suitedash.com">SuiteDash.com</a> is the home of SuiteDash, an all-in-one SaaS tool for all of your Client Portal, CRM, Sales Funnel, Appointment Scheduling, Proposals, File Sharing, Project Management, Invoicing, eSigning, LMS, Email Marketing needs.</p>
<h3 style="margin-bottom:20px;">IMPORTANT :: What is the SuiteDash WordPress Plugin?</h3>
<p>This plugin allows you to add a customized login functionality directly on your WordPress site, and have that login tied-in to your SuiteDash account. This allows you, your staff, and your clients to login to your SuiteDash environment directly from your site, without ever needing to visit the SuiteDash login page.</p>
<p>SuiteDash customers with a Custom URL set up are able to integrate their custom URLs with this plugin!</p>

== Installation ==

This section describes how to install the SuiteDash WordPress plugin and get it working.
<ul>
<li>Upload the newest version using 'Add new' Plugin (WordPress -> Plugins -> Add New), Upload via zip, Upload, and then Activate - OR - Unzip the files on your local machine and upload via FTP to the /plugins/ directory.. then activate from the plugins list.</li>
</ul>
